## Financial KPI Dashboard (Excel)

## Project Overview
This project demonstrates the development of a **Financial KPI Dashboard** using **Microsoft Excel**, designed to monitor and analyze key financial metrics such as **Revenue, Expenses, Profit/Loss, and Maximum Revenue**.

The dashboard leverages **Pivot Tables connected to the Excel Data Model (Power Pivot)** to provide a structured, dynamic, and executive-level view of financial performance. The primary goal of this project is to transform raw financial data into **clear and actionable insights** through effective data modeling and professional dashboard design.

## Objectives
- Structure and prepare financial data for analysis  
- Build Pivot Tables using the Excel Data Model  
- Create dynamic KPI cards for key financial metrics  
- Enable time-based filtering (e.g., monthly analysis)  
- Design a clean, professional, and presentation-ready dashboard  
- Ensure scalability and stability during data refresh  

## Tools & Technologies Used
- Microsoft Excel  
- Power Pivot / Data Model  
- Pivot Tables  
- Pivot Charts  
- GETPIVOTDATA for KPI extraction  
- Dashboard design and formatting  
- Slicers for interactive filtering  

## Data Preparation & Modeling
The following steps were performed during data preparation and modeling:

- Reviewed and organized raw financial data for analysis  
- Ensured correct data types for dates and numerical values  
- Built relationships between tables inside the Data Model  
- Created measures for Revenue, Expenses, and Profit/Loss  
- Validated calculations and aggregations across different time periods  

## Dashboard Features

### KPI Cards
The dashboard includes KPI cards that display the following metrics:
- Total Revenue  
- Total Expenses  
- Profit / Loss  
- Maximum Revenue  

Each KPI card is dynamically linked to Pivot Tables using **GETPIVOTDATA**, ensuring accuracy and consistency even after data refreshes.

### Time-Based Analysis
- Month-level filtering using slicers  
- Automatic updates across all KPI cards and visuals  

### Interactive Design
- Dedicated KPI Cards sheet for calculations  
- Dashboard sheet referencing helper KPI values  
- Clean layout optimized for reporting and presentations  

## Key Insights
- Financial performance can be monitored quickly using KPI cards  
- Month-based filtering allows efficient comparison of financial results  
- Profit and loss trends become immediately visible without manual calculations  
- The dashboard supports faster and more informed decision-making  

## Best Practices Applied
- Clear separation between the **analysis layer** (Pivot Tables and KPI Cards) and the **presentation layer** (Dashboard)  
- Avoided hard-coded values to maintain flexibility and scalability  
- Used Data Model measures instead of manual formulas  
- Ensured stable dashboard behavior during Pivot Table refresh  

## Business Value
This dashboard can be used for:
- Monthly financial performance tracking  
- High-level management and executive reporting  
- Entry-level financial and data analysis workflows  

It reduces manual reporting effort while improving clarity, consistency, and usability in financial monitoring.
